# Exp 8: 论文原生奖励深度修正实验计划

**背景**：在 Exp 7.3 中，我们发现由于防爆常数 $c$ 削弱了远端引力，且惩罚项权重过高，导致 Agent 陷入了“匀速绕圈”的局部最优。本实验旨在严格遵循论文的“三重密集引力”思想进行修复。

## 实验 8.1：奖励公式极速修复 (当前正在运行)
**目标**：验证在不修改轨迹生成算法（仍使用 Bézier）的前提下，仅通过释放引力奇点和平衡惩罚权重，能否打破绕圈的局部最优。

**修改内容**：
1. **释放引力**：去掉 $c=0.1$，改用 `torch.clip(1.0 / (x + 1e-5), 0.0, 20.0)`。最大奖励从 10 提升到 20，且在近距离梯度更陡峭。
2. **降低推盘惩罚 ($r_p$)**：权重从 3.0 骤降至 **0.5**。允许 Agent 在初期大胆试错，即使撞到托盘也不会被重罚。
3. **降低动作惩罚 ($r_a$)**：权重从 0.5 降至 **0.1**。释放 Agent 的转向能力，让它敢于打方向盘纠偏。
4. **回调停滞惩罚 ($r_{ini}$)**：权重从 20.0 降回 **5.0**。配合 $r_d > 0.3$ 的距离门槛，减少它在远处的“移动焦虑”。

**预期结果**：Agent 应该会放弃绕圈，重新开始向托盘发起冲锋。初期可能会有大量推盘行为（因为 $r_p$ 降低了），但随着 $r_{cd}$ 和 $r_{c\psi}$ 的引导，它应该能逐渐学会对准。

## 实验 8.2：Clothoid 轨迹重构 (待执行)
**目标**：如果 8.1 实验中 Agent 虽然敢靠近了，但依然无法完美对准（因为 Bézier 曲线在远端的朝向引导不够平滑），我们将重写轨迹生成逻辑。

**计划修改内容**：
1. 将 `_build_reference_trajectory` 中的三次 Bézier 曲线替换为 **Clothoid 曲线（回旋曲线）近似**。
2. 确保曲率从直线平滑过渡到弯道，为后轮转向叉车提供完美的 $r_{c\psi}$ 梯度场。
3. 探索在 RSL-RL 中加入 $L_{bound} = ||\mu(o_t)||$ 损失项，彻底替代 $r_a$。
