# Exp 8.1 实验结果与深度反思：引力释放与末端引导失效

**日期**：2026-03-19
**实验分支**：`exp/vision_cnn/exp8_paper_native_refine` (Run: `exp8_1_clip20_rp0.5_ra0.1`)

## 1. 实验核心配置（基于论文启发的修正）
本实验旨在验证：在不改变 Bézier 轨迹生成算法的前提下，仅通过**释放绝对状态奖励的引力**和**降低惩罚权重**，能否打破 Exp 7.3 中“1.9m 匀速绕圈”的局部最优。

- **释放引力奇点**：去掉了防爆常数 $c=0.1$，将 $r_d, r_{cd}, r_{c\psi}$ 改为 `torch.clip(1.0 / (x + 1e-5), 0.0, 20.0)`。最大单项奖励从 10 提升到 20。
- **降低推盘惩罚 ($r_p$)**：权重从 3.0 骤降至 **0.5**，允许初期试错。
- **降低动作惩罚 ($r_a$)**：权重从 0.5 降至 **0.1**，释放转向能力。
- **停滞惩罚 ($r_{ini}$)**：权重降回 **5.0**（保留 $r_d > 0.3$ 的距离门槛）。

## 2. 实验结果现象：从“勇敢冲锋”到“久攻不下后的退缩”

实验运行至约 450 代，呈现出极其经典的“探索-受挫-退缩”三阶段演化：

### 阶段一：引力爆发，勇敢冲锋 (Iter 0 ~ 200)
- **现象**：去掉 $c=0.1$ 后，近距离的奖励变得极其丰厚（`R_plus` 飙升至 30+）。Agent 像飞蛾扑火一样被托盘吸引，`dist_front` 迅速下降并稳定在 **0.35m** 左右。
- **姿态**：`yaw_deg_mean` 改善至 **23.2°**，不再像上个实验那样横着开。
- **结论**：**释放引力和降低惩罚的策略完美起效！** Agent 成功走出了“绕圈逃避”的坑，敢于在托盘前进行贴脸试错。

### 阶段二：末端引导失效，无效摩擦 (Iter 200 ~ 350)
- **现象**：虽然 Agent 靠得很近（0.35m~0.5m），但 `frac_inserted`（插入率）始终卡在 10%~15% 的瓶颈，无法进一步提升。
- **原因**：**Bézier 曲线的致命缺陷暴露。** 在最后半米的精细对准阶段，Bézier 曲线的曲率变化不够平滑，无法提供精确的 $r_{c\psi}$（偏航角）引导。Agent 只能在 20°~30° 的偏航角下，通过硬推托盘（`pallet_disp_xy` ~ 0.5m）来“蹭”取微小的插入奖励。

### 阶段三：寻找新舒适区，再次退缩 (Iter 350 ~ 450+)
- **现象**：在经历了漫长的“无效贴脸”后，Agent 发现贴脸不仅插不进去，还要不断微调方向盘（扣 $r_a$）且偶尔撞击（扣 $r_p$）。
- **退缩**：它的 Value Network 重新计算了投入产出比，决定退回到 **1.0m** 左右的距离，并再次将偏航角放大到 **59°**。
- **新局部最优**：在 1.0m 处横着开，既能拿到不错的 $r_{cd}$（轨迹距离）奖励，又能减少转向动作和撞击风险。

## 3. 深度反思与下一步计划 (Exp 8.2)

**Exp 8.1 已经完成了它的历史使命。** 它证明了：
1. **绝对状态奖励 $1/x$ 是可行的**，前提是不能用常数 $c$ 去“盖帽”它的引力奇点。
2. **惩罚项必须克制**，在 Agent 学会完美对准之前，过高的 $r_p$ 会直接扼杀探索。

**当前的唯一瓶颈：缺乏精确的末端引导。**
当 Agent 勇敢地冲到托盘前 0.5m 时，它需要一条曲率完美、平滑过渡的参考线来告诉它“现在该把方向盘回正了”。Bézier 曲线做不到这一点。

**下一步行动 (Exp 8.2)：全面引入 Clothoid 曲线**
我们将停止当前的训练，执行之前准备好的 `test_clothoid_v3.py` 方案：
1. 将 `env.py` 中的参考轨迹生成逻辑从 Bézier 替换为 **三次 Hermite 样条 (Cubic Hermite Spline)**。
2. 通过控制切线长度 $L$，生成一种完美近似 **Clothoid 曲线（回旋曲线）** 的大肚子平滑轨迹。
3. 利用 Clothoid 曲线“曲率线性变化”的特性，为后轮转向的叉车铺设一条从远端到近端、引导极其明确的奖励高速公路。
