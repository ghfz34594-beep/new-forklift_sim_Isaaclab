# 实验 6.0b：微步泛化与核弹奖励的震荡与回档 (2026-03-15)

## 1. 实验背景与目标
在 **Exp 5.9** 中，我们通过“核弹奖励 (rg=5000) + Early Stopping”在极窄的初始范围（±0.05m, ±2°）内取得了史诗级的成功（75% 成功率，完美姿态）。
但在随后的 **Exp 6.0** 中，我们尝试将初始范围大幅放宽到（±0.5m, ±15°），导致了灾难性遗忘，模型完全崩溃，变成了“推土机”。

因此，**Exp 6.0b** 采取了“微步泛化”策略：
*   **初始范围**：从极窄的（±0.05m, ±2°）微调到 **（±0.1m, ±5°）**。
*   **奖励设置**：恢复 `alpha_4 = 5000.0`（核弹奖励），并移除空间衰减，恢复全局姿态引导。
*   **起点**：加载 Exp 5.9 的“神级” Checkpoint (`model_836.pt`)。
*   **目标**：在不发生灾难性遗忘的前提下，让模型适应稍微宽一点的初始状态。

## 2. 实验过程与现象观察

本次实验跑了约 2300 代（从 836 代恢复，实际运行约 1500 代），展现了强化学习在泛化过程中极其经典的**策略震荡（Policy Oscillation）**现象。

### 阶段一：适应与反弹（~1100代）
*   **现象**：刚开始时，模型对新范围有些不适应，成功率在 30%~40% 之间震荡，偏航角（yaw）一度飙升到 23°。
*   **纠错**：由于我们保留了严格的姿态惩罚（`alpha_2=5.0`, `alpha_3=5.0`），模型在尝试“斜着硬挤”后受到了严厉惩罚。在 1000 代左右，yaw 角被成功压制回 10° 左右。

### 阶段二：巅峰时刻（1400代 ~ 1500代）—— 【高光时刻】
*   **现象**：在第 **1443 代**，成功率 (`rg`) 历史性地突破了 50% 大关，达到 **51.56%**！
*   **姿态表现**：在随后的 1445-1446 代，虽然成功率略微回落到 45% 左右，但偏航角（yaw）达到了惊人的 **6.5°**，推盘位移（disp）控制在 0.78m。
*   **结论**：这证明了“微步泛化 + 核弹奖励”策略是**完全可行**的。模型有能力在这个新范围内找到高成功率且姿态完美的解。

### 阶段三：统计学假象与探索迷失（1600代 ~ 2000代）
*   **现象**：日志中多次出现 `rg = 0.5000`（如 1697, 1703, 1798, 1953 代），但伴随的却是高达 **15°~19° 的平均偏航角**。
*   **深度解析（平均值的掩盖）**：这说明在 64 个并行环境中，有 32 个环境成功了（且姿态必然 < 15°），而另外 32 个失败的环境（通常是出生在极端边缘位置）完全崩溃，采用了“推土机”策略，产生了 25° 甚至 30° 以上的极大偏差，从而拉高了整体平均值。

### 阶段四：策略退化与泥潭（2000代 ~ 2337代）
*   **现象**：模型似乎在探索中“走火入魔”。在 2323 代，平均 yaw 飙升至 **29.6°**，横向误差达 0.48m，推盘位移飙升至 1.47m。成功率跌至 23%~30%。
*   **原因**：模型在边缘困难位置频繁受挫，最终放弃了精细的姿态调整，退化成了纯粹的“闭眼推土机”策略。由于这种策略拿不到 5000 分大奖，且吃满了推盘和姿态惩罚，Critic 网络陷入迷茫，导致整体策略崩溃。

## 3. 结论与下一步行动 (Exp 6.0c)

**结论**：
Exp 6.0b 并非完全失败。它在 1450 代左右证明了泛化的可行性，但随后因为在困难边缘位置的探索失败，导致了整体策略的退化。

**行动计划（止损回档）**：
1.  **停止训练**：已在 2337 代手动终止，避免继续浪费算力。
2.  **提取巅峰权重**：提取了表现最好的 **`model_1450.pt`**（rg>50%, yaw~6.5° 附近）。
3.  **生成可视化视频**：目前正在使用 `model_1450.pt` 生成视频，以直观分析其在巅峰状态下的表现和潜在的薄弱环节。
4.  **开启 Exp 6.0c**：
    *   基于 `model_1450.pt` 重新开始。
    *   保持（±0.1m, ±5°）的范围不变。
    *   **策略微调**：考虑稍微降低推盘惩罚（`alpha_5` 从 3.0 降至 2.0），或者引入动态的姿态容忍度，防止模型在边缘位置因为过度惩罚而“破罐子破摔”。
