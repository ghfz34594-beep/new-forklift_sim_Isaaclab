# Exp 6.0 失败反思与“微步横向扩展”策略的深度推演

**日期**：2026-03-15

## 一、 Exp 6.0 为什么会遭遇“灾难性遗忘”？
在 Exp 6.0 中，我们将初始横向偏差放宽到 `±0.5m`，偏航角放宽到 `±15°`，并引入了“空间衰减 Shaping”，同时将大奖降回 200。结果导致 Agent 彻底崩溃，成功率归零，变成了“歪着身子的推土机”。

**核心原因剖析**：
1.  **探索空间的维度爆炸**：从 `±0.05m` 到 `±0.5m`，看似只是乘了 10，但在高维状态空间中，Agent 面临的未知状态呈指数级增长。它之前学到的“闭眼踩油门”策略在 99% 的新状态下都会导致撞墙。
2.  **奖励信号的“断层” (Credit Assignment Problem)**：
    *   **远处的迷茫**：因为“空间衰减”，在 3 米外它得不到姿态奖励，只能靠距离奖励 `r_dist` 盲目往前开。
    *   **近处的绝望**：等它开到 1 米内（姿态奖励开始生效）时，它往往已经处于一个极其糟糕的姿态（比如偏航 30°，横向偏了 0.5m）。在这么近的距离下，物理引擎根本不允许它再做大幅度的转向调整（会直接撞在托盘上）。
    *   **大奖的消失**：因为姿态太差，它永远触发不了 `rg`，200 分的大奖形同虚设。
3.  **结论**：**步子迈得太大，扯到了蛋**。Agent 在远处没有得到正确的引导，导致近处陷入死局，最终在一遍遍的失败中，把 5.9 辛苦学来的“完美姿态”权重给洗掉了。

## 二、 “微步横向扩展”策略是否能解决问题？

**策略内容**：
1.  退回 5.9 神级起点 (`model_836.pt`)。
2.  极小步长放开：横向 `±0.1m`，偏航 `±5°`。
3.  保持核弹奖励：`alpha_4 = 5000`。
4.  移除空间衰减，恢复全局姿态引导。

**深度推演（为什么这可能是对的）**：
1.  **保持在“舒适区”边缘探索**：`±0.1m` 和 `±5°` 刚好在 Agent 原有能力的边缘。它偶尔用旧的“直行”策略会失败，但只要稍微打一点点方向盘，就能再次触发 5000 分的大奖。这种“稍微努力一下就能够到”的难度，是强化学习中最容易产生有效梯度的区间。
2.  **核弹奖励的“锚定效应”**：5000 分的奖励就像一座灯塔。只要它在新的起跑线上偶然成功一次，巨大的正向梯度就会瞬间告诉它：“对，在偏 10 厘米的时候，你应该这样打方向盘”。这能有效对抗遗忘。
3.  **全局姿态引导的必要性**：移除空间衰减意味着，Agent 在 3 米外一出生，就会因为 `r_lat` 和 `r_yaw` 的扣分（相对满分而言）而感到“难受”。它会被迫在**距离托盘还很远、物理空间还很宽敞**的时候，就开始主动修正姿态。等它靠近托盘时，姿态已经基本摆正了。

**潜在风险与应对**：
*   **风险**：恢复全局姿态引导后，它会不会又在远处“原地摆 Pose”？
*   **应对**：有可能。但因为我们保留了 5000 分的插入大奖，只要它摆完 Pose 后往前开能拿到 5000 分，Critic 网络最终会明白“摆完 Pose 往前开”的价值远大于“只摆 Pose 不往前开”。

## 三、 结论
“微步横向扩展（Curriculum Learning for Generalization）”是目前唯一符合强化学习认知规律的破局之道。我们必须像剥洋葱一样，一层一层地把难度加上去，而不是指望它能瞬间适应狂风暴雨。

如果 `±0.1m` 成功了，我们再加到 `±0.2m`，直到它能完全覆盖 `±0.5m` 的真实场景。
